Bad apple for agon-light
=======================

Bad apple for the agon light built using z88dk.


Place badapple.bin and badapple.txt in the same directory.

```
CD demos
LOAD badapple.bin
RUN
```

Agon Light https://github.com/TheByteAttic/AgonLight
